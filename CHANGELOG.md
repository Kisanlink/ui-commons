# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-13

### Added
- Initial implementation of @kisanlink/ui-commons
- Button component with multiple variants (primary, secondary, success, error, warning, info, ghost, outline)
- Input component with icon support and validation states
- Badge component for status indicators
- Card component with header, body, and footer sub-components
- ThemeProvider for light/dark mode support
- Design token system (colors, typography, spacing)
- CSS Modules styling approach
- Comprehensive TypeScript types
- Test infrastructure with Vitest
- Storybook documentation
- Build system with Vite
